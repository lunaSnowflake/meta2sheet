Privacy Policy for Meta2Sheet

Effective Date: 2025-04-12

Meta2Sheet is a Chrome extension that enables users to easily transfer data from Metabase dashboards to Google Sheets. We are committed to protecting your privacy and being transparent about the permissions we request and how we use them.

Information We Collect

Meta2Sheet does not collect, transmit, or store any personally identifiable information (PII), such as names, email addresses, or passwords.

Permissions Explained

  • activeTab: Used to interact with the current Metabase dashboard page for extracting visible filters.
  • scripting: Required to inject scripts into the current tab for reading Metabase dashboard parameters.
  • storage: Used to locally store user preferences like selected Google Sheet and tab, to improve the user experience.
  • identity: Used to authenticate the user's Google account for accessing Google Sheets via OAuth2.
  • cookies: This extension reads cookies to securely access the active session on Metabase (e.g., metabase.SESSION) in order to extract dashboard data the user is authorized to view. It also uses cookies if required by Google APIs to maintain the authenticated session when transferring data to Google Sheets. No cookie data is stored or transmitted to any third-party servers.
  • host permissions: Required to access and interact with Metabase dashboards that may be hosted on various custom domains. Only interacts with Metabase URLs the user actively navigates to.

Data Usage

All data processing (such as reading dashboard filters and sending them to Google Sheets) happens locally in your browser or through secure calls to Google APIs. No data is transmitted to any external servers operated by us.

Data Responsibility Disclaimer

Meta2Sheet allows users to transfer data between Metabase and Google Sheets. Depending on how the extension is used, existing data in the destination spreadsheet may be modified or replaced. Users are solely responsible for managing and backing up their data. Meta2Sheet does not accept liability for any data loss or unintended changes resulting from the use of this extension.

Third-Party Services

We use Google APIs (such as Google Sheets API) to facilitate data transfer. All communication with Google services is securely handled using industry-standard OAuth2.
